# Projeto de Portfólio – Análise de Dados com Python

Este é meu primeiro projeto de portfólio na área de dados.  
O objetivo é demonstrar habilidades básicas de programação em Python e análise exploratória de dados.

## Objetivo do projeto
Explorar um conjunto de dados real, aplicar operações simples de análise e gerar visualizações.

## Ferramentas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Etapas realizadas
- Carregamento de dados públicos em um notebook Python
- Exploração inicial dos dados (dimensão, colunas, estatísticas básicas)
- Cálculo de métricas simples (exemplo: média)
- Criação de gráfico para visualização dos dados
